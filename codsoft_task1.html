Project

ENVIRONMENTAL MONITORING IN PARKS

This project aims to address the problem faced by park visitors who lack real-time environmental data for planning outdoor activities. By implementing an IoT-based solution, it involves deploying sensors like DHT11 or DHT20 to monitor humidity and temperature in parks. Data is processed using an Arduino connected to an ESP8266 module for cloud connectivity. Visitors receive notifications via a mobile app when conditions are suitable for outdoor plans, improving visitor satisfaction and aiding park management in organizing activities effectively
